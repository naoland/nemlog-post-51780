# 簡単プログラミング！BTCの複数取引所の現在価格を表示しよう（Node.js 編）

## はじめに

みなさん、こんにちは。

今回は複数の取引所の、現在のBTCの価格を表示してみます。

3つの取引所（Zaif、Bitflyer、BitBank）からBTC/JPY通貨ペアのティッカー情報を取得し、それぞれの現在価格をコンソールに表示します。
今回も動作確認はGitpodを使用します。
